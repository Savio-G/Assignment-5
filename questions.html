What is Dom and What is the purpose of DOM?

DOM is document object model. The purpose of DOM is to present the webpage to us as objects and nodes which enables different programming languages to communicate with the website

What is event bubble

Event bubble is a concept of DOM which indicates the movement of commands by the usage of parent and childnodes where parent nodes has the priority
